# Dashboard Analisis Penyewaan Sepeda
Dashboard ini merupakan proyek akhir dari materi Fundamental Analisis Data yang bertujuan untuk memberikan insight mengenai performa penyewaan sepeda yang dipengaruhi oleh berbagai faktor. Dashboard ini menggunakan Python dan library Streamlit.

## Setup environment
```
conda create --name main-ds
conda activate main-ds
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
